# DialogueGame

DialogueGame is a fully text based game in which the player is experiencing 
a story, the main mechanics are reading dialogue as part of the story and 
responding with either more dialogue or actions.

## Technologies
- Language: C#
- Framework: .NET 10
- IDE: Visual Studio

## Current Features
- Typewriter text effect
- Player name input at the start
- Branching dialogue with multiple choices
- Inventory system — items picked up affect later scenes
- Multiple scenes with unique descriptions

## Objectives
- Learn GitHub
- Learn new C# concepts

## How to Run
- Clone the repo
- Open the `.sln` file in Visual Studio
- Hit Run
